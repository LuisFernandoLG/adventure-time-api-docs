You can access the list of characters by using the `/characters` endpoint.
<CodeEndpoint path="/characters"/>

## Response Schema
| key            | type                                                    | description                                           |
| -------------- | ------------------------------------------------------- | ----------------------------------------------------- |
| items           | [character](/docs/characters/getone/#character-model)[] | array of characters.                                  |
