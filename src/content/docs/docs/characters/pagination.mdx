```bash
https://adventure-time-api-lg.vercel.app/apiV1/characters/paginated
```

## Response Schema

| key            | type                                                    | description                                           |
| -------------- | ------------------------------------------------------- | ----------------------------------------------------- |
| current_page   | int                                                     | current page fetched.                                 |
| data           | [character](/docs/characters/getone/#character-model)[] | array of characters.                                  |
| next_page_url  | string                                                  | link to get next `n` elements                         |
| first_page_url | string                                                  | URL for the first page of the paginated data          |
| from           | int                                                     | starting index of the current page's data             |
| last_page      | int                                                     | number of the last available page                     |
| last_page_url  | string                                                  | URL for the last page of the paginated data           |
| links          | array                                                   | an array of objects containing link information       |
| links.url      | string                                                  | URL associated with the link                          |
| links.label    | string                                                  | label or text associated with the link                |
| links.active   | bool                                                    | indicates whether the link is currently active or not |

## Params
There are special params for pagination.
### PerPage
If you don't want always get `10` items, you can set the `perPage` param to the endpoint:
```
https://adventure-time-api-lg.vercel.app/apiV1/characters/paginated?perPage=2
```
### page
with the param `page` you can get the specific page you want.
```
https://adventure-time-api-lg.vercel.app/apiV1/characters/paginated?perPage=2
```
